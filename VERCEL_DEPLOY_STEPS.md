## ❌ Common Issues & Fixes

### Issue: "Repository not found"
```
Solution:
1. Make sure you signed into Vercel with GitHub
2. Check GitHub - code is pushed
3. Try refreshing Vercel page
4. Re-authorize GitHub if needed
```

### Issue: "Build failed"
```
Solution:
1. Check Vercel logs for error
2. The error will be specific
3. Most likely: missing node_modules
4. Vercel auto-installs, so this is rare
```

### Issue: "Blank page after deploy"
```
Solution:
1. Clear browser cache
2. Hard refresh (Ctrl+Shift+R)
3. Check console (F12) for errors
4. Check Vercel logs
```

### Issue: "MetaMask not working"
```
Solution:
1. Install MetaMask extension
2. Refresh page
3. Click Connect
4. Should work!
```
